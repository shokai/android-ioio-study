IOIO LED Blink
==============
blink LED [http://shokai.org/blog/archives/5625](http://shokai.org/blog/archives/5625)


Build and Install
-----------------

use Eclipse+ADT, or

    android update project --path `pwd`
    ant debug
    adb install -r bin/MainActivity-debug.apk


Uninstall
---------

    adb uninstall org.shokai.ioio.ledblink
